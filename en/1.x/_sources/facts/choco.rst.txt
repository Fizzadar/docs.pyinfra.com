Choco Facts
-----------

:code:`choco_packages`
~~~~~~~~~~~~~~~~~~~~~~

.. code:: python

    host.fact.choco_packages


Returns a dict of installed choco (Chocolatey) packages:

.. code:: python

    {
        'package_name': ['version'],
    }



:code:`choco_version`
~~~~~~~~~~~~~~~~~~~~~

.. code:: python

    host.fact.choco_version


Returns the choco (Chocolatey) version.
