Apt Facts
---------

:code:`AptKeys`
~~~~~~~~~~~~~~~

.. code:: python

    host.get_fact(AptKeys)


:code:`AptSources`
~~~~~~~~~~~~~~~~~~

.. code:: python

    host.get_fact(AptSources)


Returns a list of installed apt sources:

.. code:: python

    [
        {
            'type': 'deb',
            'url': 'http://archive.ubuntu.org',
            'distribution': 'trusty',
            'components', ['main', 'multiverse'],
        },
    ]
