Windows Facts¶

WindowsAliases¶

host.get_fact(WindowsAliases)

Returns the Windows aliases.

WindowsBios¶

host.get_fact(WindowsBios)

Returns the BIOS info.

WindowsComputerInfo¶

host.get_fact(WindowsComputerInfo)

Returns the Windows info.

WindowsDate¶

host.get_fact(WindowsDate)

Returns the current datetime on the server.

WindowsHome¶

host.get_fact(WindowsHome)

Returns the home directory of the current user.

WindowsHostname¶

host.get_fact(WindowsHostname)

Returns the current hostname of the server.

WindowsHotfixes¶

host.get_fact(WindowsHotfixes)

Returns the Windows hotfixes.

WindowsInstallerApplications¶

host.get_fact(WindowsInstallerApplications)

Returns the Windows installer applications.

WindowsLastReboot¶

host.get_fact(WindowsLastReboot)

Returns the date and time of the last reboot.

WindowsLocalDrivesInfo¶

host.get_fact(WindowsLocalDrivesInfo)

Returns the Windows local drives info.

WindowsLocalGroups¶

host.get_fact(WindowsLocalGroups)

Returns a list of groups on the system.

WindowsLoggedInUserInfo¶

host.get_fact(WindowsLoggedInUserInfo)

Returns the Windows user logged in info.

WindowsLogonSessionInfo¶

host.get_fact(WindowsLogonSessionInfo)

Returns the Windows user logon session info.

WindowsNetworkConfiguration¶

host.get_fact(WindowsNetworkConfiguration)

Returns the Windows network configuration.

WindowsOs¶

host.get_fact(WindowsOs)

Returns the OS name according to systeminfo.

WindowsOsVersion¶

host.get_fact(WindowsOsVersion)

Returns the OS version according to systeminfo.

WindowsProcesses¶

host.get_fact(WindowsProcesses)

Returns the Windows processes.

WindowsProcessors¶

host.get_fact(WindowsProcessors)

Returns the processors info.

WindowsService¶

host.get_fact(WindowsService, name)

Returns info about a Windows service.

WindowsServices¶

host.get_fact(WindowsServices)

Returns the Windows services.

WindowsSystemType¶

host.get_fact(WindowsSystemType)

Returns the system type according to systeminfo.

WindowsWhere¶

host.get_fact(WindowsWhere, name)

Returns the full path for a command, if available.
