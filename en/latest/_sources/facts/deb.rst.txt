Deb Facts
---------

:code:`deb_package`
~~~~~~~~~~~~~~~~~~~

.. code:: python

    host.fact.deb_package(name)


Returns information on a .deb archive or installed package.



:code:`deb_packages`
~~~~~~~~~~~~~~~~~~~~

.. code:: python

    host.fact.deb_packages


Returns a dict of installed dpkg packages:

.. code:: python

    {
        'package_name': ['version'],
    }
