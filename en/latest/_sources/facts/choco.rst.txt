Choco Facts
-----------

:code:`ChocoPackages`
~~~~~~~~~~~~~~~~~~~~~

.. code:: python

    host.get_fact(ChocoPackages)


Returns a dict of installed choco (Chocolatey) packages:

.. code:: python

    {
        'package_name': ['version'],
    }



:code:`ChocoVersion`
~~~~~~~~~~~~~~~~~~~~

.. code:: python

    host.get_fact(ChocoVersion)


Returns the choco (Chocolatey) version.
