Help & Support¶

Need a hand with using pyinfra or have some questions? You can find official pyinfra communities below:

  • Chat in the pyinfra Matrix room
  • Ask questions on Stack Overflow (be sure to use the pyinfra tag!)

I think I’ve got a bug!¶

  • Please have a quick scan of the the compatibility page to see if the issue is referenced there
  • Check the issue list to see if anyone else has this issue
  • Submit a new bug report

I have an idea!¶

Please consider submitting a feature request.

I’d like to contribute!¶

Fantastic! Please see the contributing docs.
