Pip Facts
---------

:code:`Pip3Packages`
~~~~~~~~~~~~~~~~~~~~

.. code:: python

    host.get_fact(Pip3Packages, pip=None)


:code:`PipPackages`
~~~~~~~~~~~~~~~~~~~

.. code:: python

    host.get_fact(PipPackages, pip=None)


Returns a dict of installed pip packages:

.. code:: python

    {
        'package_name': ['version'],
    }
