Deb Facts
---------

:code:`DebPackage`
~~~~~~~~~~~~~~~~~~

.. code:: python

    host.get_fact(DebPackage, name)


Returns information on a .deb archive or installed package.



:code:`DebPackages`
~~~~~~~~~~~~~~~~~~~

.. code:: python

    host.get_fact(DebPackages)


Returns a dict of installed dpkg packages:

.. code:: python

    {
        'package_name': ['version'],
    }
