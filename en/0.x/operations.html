Operations IndexÂ¶

Operations are used to describe changes to make to systems in the inventory. Use them to define state and pyinfra will make any necessary changes to reach that state. Operations can be executed via the command line or by creating a deploy.

All operations accept a set of global arguments and are namespaced as different modules:
